# 60 Days 60 Tools

A 60-day build series for shipping small, useful tools.

## Goal

Build one focused tool per day for 60 days. Each tool should solve a clear problem, be easy to run, and include enough notes for someone else to understand the idea.

## Structure

This repository is ready for tools to be added as the series progresses. A suggested layout is:

```text
tools/
  day-01-tool-name/
    README.md
    src/
```

## Development

Add each tool in its own folder under `tools/` and include a short README with:

- what the tool does
- how to run it
- any required environment variables
- what changed or was learned while building it
