# Tool 01: AI Challenge Tracker

Status: Live

## What It Does

A public dashboard to track every AI tool built during the 60-day challenge, including links to code, demos, lessons, skills used, and daily write-ups.

## Why This Exists

Before building 60 tools, the challenge needs a clear public home. This first tool acts as the challenge operating system: it shows the current progress, makes the journey easy to follow, and creates a structure that can grow day by day.

## Skills

- Next.js / React
- portfolio design
- data modelling
- public build logs
- AI-assisted workflow planning
